# Verse of the Day
A small bash script to print Bible Gateway's Verse of the Day in KJV

## Installation

#### Download and execute the script:

```bash
$ wget https://github.com/tmiland/votd/raw/master/votd.sh
$ chmod +x votd.sh
$ ./votd.sh
```

#### Optionally

 ```bash
$ ln -s /path/to/votd.sh /usr/bin/votd
$ votd
```
